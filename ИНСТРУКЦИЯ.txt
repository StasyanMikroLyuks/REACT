установить node js
1) npm init react-app ... 
где ... название папки приложения
2) npm install react-router
3) npm install -D tailwindcss
4) npx tailwindcss init
5) Добавить строки 
@tailwind base;
@tailwind components;
@tailwind utilities;
в .../src/index.css
6) cd ...
7) npm run start
изменять App.js = главная страница
